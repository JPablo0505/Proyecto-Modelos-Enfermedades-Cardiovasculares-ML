# Predicción de Enfermedades Cardiovasculares mediante Machine Learning

## Descripción del Proyecto
Sistema de predicción de enfermedades cardiovasculares (ECV) implementando y comparando 5 modelos de Machine Learning sobre el Cardiovascular Disease Dataset de Kaggle. Este proyecto fue desarrollado como trabajo final para el curso de Modelos y Simulación II.

## Objetivos
- Desarrollar un sistema predictivo robusto para ECV
- Comparar el desempeño de diferentes algoritmos de ML
- Identificar las variables más relevantes para la predicción
- Evaluar técnicas de reducción de dimensionalidad

## Dataset

### Cardiovascular Disease Dataset
**Fuente**: [Kaggle](https://www.kaggle.com/datasets/sulianova/cardiovascular-disease-dataset)  
**Muestras**: 70,000 pacientes  
**Variables**: 12 variables clínicas y de hábitos

## Ejecución
1. Abrir `proyecto_modelos_ii.ipynb` en Google Colab
2. Ejecutar celdas en orden
3. Los resultados se generan automáticamente

## Modelos Implementados
- Random Forest
- Support Vector Machine (SVM)
- Red Neuronal Artificial
- Regresión Logística
- K-Nearest Neighbors (K-NN)

## Resultados
Mejor modelo: **Random Forest** (F1-Score: 0.7381, AUC: 0.7998)
